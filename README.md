# Neon Runner Quest

Neon Runner Quest is a dependency-free HTML5 Canvas action game. Open `index.html` in a browser and play directly.

## Gameplay

- Clear five waves of enemies.
- Defeat the boss core on wave 5.
- Pick up health, energy, and shield drops.
- Keep moving, use dash invulnerability, and chain attacks for higher damage.
- The high score is stored in `localStorage` with the key `neonRunnerQuestHighScore`.

## Controls

- Move: `WASD` or arrow keys
- Attack: `J`, mouse click, or the touch attack button
- Dash: `K`, `Shift`, or the touch dash button
- Pause: `P`
- Restart: `R`

## Files

- `index.html`: Complete game, styles, and scripts in one static file.
- `assets/`: Hand-built SVG sprites for the player, enemies, boss, and pickups.
- `README.md`: Project notes and controls.

## Run

Double-click `index.html`, or drag it into any modern desktop browser.
